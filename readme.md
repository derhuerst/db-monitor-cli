## Installing

```shell
npm install db-monitor-cli
```


## Usage

```shell
monitor [--interval milliseconds] station-id [...]
```

Writes [ndjson](http://ndjson.org/) in [the format of `db-monitor`](https://github.com/derhuerst/db-monitor) to stdout.
